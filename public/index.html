Anonyme Lehrveranstaltungsevaluation

Ein browserbasiertes Werkzeug für eine Zwischenevaluation nach der Methode des Teaching Analysis Poll (TAP). Studierende geben anonymes Feedback; die Auswertung – inklusive einer KI-gestützten Verdichtung der Freitextantworten – läuft ausschließlich im Browser der Lehrperson.

So funktioniert es

  1. Die Lehrperson legt eine Evaluation an (Name der Veranstaltung + Semester).
  2. Studierende öffnen den Link bzw. scannen den QR-Code und antworten anonym.
  3. Die Antworten laufen direkt in den Browser der Lehrperson – der Server speichert nichts.
  4. Zum Abschluss: Diagramme der Likert-Skalen, KI-Zusammenfassung der Freitexte, drei TAP-Abschnitte und vier überarbeitbare Schlussfolgerungen als Bericht.

Datenschutz im Kern

  • Anonym: keine Anmeldung, keine personenbezogenen Daten, keine Cookies.
  • Kein Server-Speicher: Antworten werden live an den Browser der Lehrperson weitergeleitet und nur dort (im localStorage) gehalten.
  • Synchron: die Befragung ist nur aktiv, solange das Dashboard der Lehrperson geöffnet ist.
